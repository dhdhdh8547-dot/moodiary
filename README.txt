Moodiary - 감정 기록 온라인 일기장

1. 실행 방법
- index.html 파일을 더블클릭해서 실행합니다.
- 또는 전체 폴더를 VS Code Live Server로 실행합니다.

2. 페이지 구성
- index.html: 일기 목록, 감정 필터, 검색, 통계, 샘플 데이터 불러오기
- write.html: 일기 작성 및 수정, 글자 수 카운트, 실시간 미리보기
- detail.html: 일기 상세 보기, 수정, 삭제, 인쇄

3. 주요 기능
- localStorage를 활용한 일기 저장
- 제목/내용 입력 검사
- 감정별 필터링
- 검색 기능
- 글자 수 실시간 카운트
- 다크 모드
- 모바일/태블릿/PC 반응형 레이아웃
- jQuery CDN 로드 시 일기 카드 fadeIn 효과 적용

4. 수업 내용 활용
- HTML 구조 작성
- CSS 레이아웃, position, sticky header, 말줄임 처리
- viewport meta 태그와 media query를 활용한 반응형 웹
- JavaScript DOM 선택 및 조작
- addEventListener를 활용한 이벤트 처리
- 동적 문서 객체 생성
- setInterval을 활용한 문구 변경
- localStorage 활용
